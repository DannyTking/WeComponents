# Contributing

**安装组件**

```console
npm install @tencent/wecomponents
```

**开发组件**

```console
npm start
```

**生成文档**

```console
npm run docs
```

**单元测试**

```console
npm run test
```

如果你有好的意见或建议，欢迎给我们提 Issues 或 Pull Requests。

[腾讯开源激励计划](https://opensource.tencent.com/contribution) 鼓励开发者的参与和贡献，期待你的加入。
